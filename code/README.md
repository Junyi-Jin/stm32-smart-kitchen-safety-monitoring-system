# STM32-Based Smart Kitchen Safety Monitoring System

This project is an STM32-based embedded system designed for smart kitchen safety monitoring.  
It integrates multiple sensors and control modules to detect abnormal environmental conditions such as gas leakage, flame, and temperature rise, and responds through alarms, actuator control, OLED display, and wireless communication.

## Features
- Temperature monitoring using DS18B20
- Gas detection using MQ sensor
- Flame detection
- OLED real-time display
- Audible and visual alarm
- Relay, fan, and servo control
- Wi-Fi communication using ESP8266

## Project Structure
- `code/` - STM32 firmware source code and project files
- `docs/` - system block diagram and selected software flowcharts
- `hardware/` - schematic and PCB layout
- `media/` - simulation, prototype, and functional test images

## Notes
- The mobile app shown in the demonstration images was used only for communication testing and data visualization.
- The core work of this project focuses on STM32 firmware, sensor interfacing, hardware integration, and system validation.
